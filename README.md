# 🚀 ALGONIVE INTERNSHIP TASK - 1

## 💬 Real-Time Chat Application

A full-stack real-time chat application built using **React.js, Node.js, Express, and Socket.IO**.  
This project allows users to communicate instantly with features similar to WhatsApp Web or Messenger.

---

## ✨ Features

- 🔐 User Authentication (Register & Login)
- 💬 One-to-One Private Chat
- ⚡ Real-Time Messaging (Socket.IO)
- ✍️ Typing Indicators
- 🟢 Online / Offline Status
- ✔️ Read Receipts (✓ / ✓✓)
- 📜 Message History
- 🔍 User Search
- 🎨 Clean & Responsive UI

---

## 🛠️ Tech Stack

| Layer       | Technology              |
|------------|------------------------|
| Frontend   | React.js, Axios        |
| Backend    | Node.js, Express.js    |
| Real-time  | Socket.IO              |
| Database   | SQLite3                |
| Auth       | JWT, bcrypt            |
| Styling    | CSS3                   |

---
